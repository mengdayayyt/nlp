1929年还是1989年?
巴黎—随着经济危机的深化和扩大,世界一直在寻找历史类比,帮助我们了解所发生的事情。 危机开始时,许多人将危机比喻为1982年或1973年,这令人欣慰,因为这两个日期都指典型的周期性衰退。
这种趋势要么是过度克制(欧洲 ) , 要么是扩大努力(美国 ) 。 欧洲以避免债务和捍卫欧元的名义谨慎行事,而美国则在许多方面有所作为,以免浪费一个理想的机会来实施急需的结构改革。
当然,雷曼兄弟倒塌与柏林墙倒塌毫无关系。 事实上,在表面看来,倒塌似乎是完全相反的:一堵墙倒塌象征着压迫和人为分裂,而一堵看似坚不可摧、令人放心的金融资本主义机构的倒塌则象征着一个象征着压迫和人为分裂的墙倒塌。
东西方意识形态分裂的终结和市场绝对信念的终结是历史转折点。 2009年发生的事情可能危及1989年的一些积极成果,包括欧洲和平统一和民主原则战胜民族主义,甚至仇外倾向。
1989年,自由民主战胜了苏维埃集团所赋予和推动的社会主义意识形态。 对他的许多支持者来说,正是罗纳德·里根总统故意使军备竞赛升级,将苏维埃经济推向边缘,从而充分显示了自由社会和自由市场的优势。
首先,也许最重要的是,1989年的革命以及随后苏联的崩溃结束了全球两极分化。 相比之下,2009年有可能为新形式的两极化铺平道路,但中国取代了苏联。
其次,民主和市场资本主义1989年的获胜者似乎非常明显 — — 如果比预期的要脆弱的话 — — 但随着全球危机的蔓延,2009年很难区分获胜者与失败者。 每个人似乎都是失败者,即使有些人受到的影响比其他人大。
然而,历史是不公平的,尽管美国对当今全球危机负有更大的责任,但它的出现可能优于大部分摆脱困境的国家。 情况更好,但不是孤立无援。
一种感知,比如美亚主导宇宙的形成。 从麻省理工学院的令人难以置信的媒体实验室到哈佛的数学和经济学系,亚洲人 — — 特别是中国人和印第安人 — — 到处都有,就像公元前第一个世纪雅典的罗马人:对他们学习如此多,未来几十年将战胜的人充满钦佩。
但在这个新秩序出现之前,世界可能会面临混乱的蔓延,如果不是彻底混乱的话。 比如,当数十万在海湾工作的埃及人由于石油生产国的危机被迫返回祖国时,像埃及这样一个核心和脆弱国家会发生什么情况呢?

人人享有跨大西洋贸易
华盛顿 — — 建立欧盟和美国跨大西洋贸易和投资伙伴关系的谈判正受到广泛欢迎。 英国首相卡梅伦将TTIP称为“一代后奖 ” , 引用了欧盟和美国各800亿英镑(1 255亿美元)和世界其他地区850亿英镑的潜在收益。
要想让世界厌烦地等待世界贸易组织无限的多哈贸易谈判回合结束,即使是双边贸易倡议也似乎都是一种好处,特别是正如最近《金融时报》社论指出的,“双边”覆盖了世界经济的一半。 但有一个严重的不利因素:除非欧盟和美国共同努力保护这些行为者的利益,否则这一交易可能会伤害发展中国家的出口商。
考虑到欧盟和美国的低关税 — — 平均不到5% — — 进一步优惠削减不会严重妨碍外来者。 但是,在标准方面 — — 诸如安全、健康和环境标准方面 — — 市场准入要求是残酷和二进制的:要么达到既定标准,要么不出售。
其结果是,第三国企业的选择将取决于如何制定TTIP标准:通过协调(采用共同标准)或相互承认(接受彼此达到既定标准的商品 ) 。 第一项选择将使世界各地的生产者都能利用规模经济。
即便新的标准适用于所有出口国的供应商,但合规成本通常各不相同,这意味着那些不具备达到更高标准的装备的人可能会受到影响。 20世纪90年代末,当欧盟决定统一黄曲霉毒素标准(一组有毒化合物由某些模子生产 ) , 八个成员国 — — 包括意大利、荷兰和西班牙 — — 大幅提高了国家标准,这可能导致非洲谷物、干果和坚果出口下降高达6.7亿美元。
如果该政策扩展到第三国公司,就会产生强大的自由化影响。 比如,马来西亚电视制作人可以选择遵守,比如,美国较容易获得的安全标准,然后在两个市场上销售同样的产品,从规模经济中获益,同时降低合规成本。
但是,如果TTIP将第三国公司排除在相互承认政策之外,它们相对于欧洲和美国公司的竞争力就会大幅下降。 事实上,我们的研究表明,当相互承认协议包含限制性原产地规则时,区域内部贸易增长 — — 牺牲了与其他国家的贸易 — — 并且发展中国家受害最深。
事实上,对欧盟先前的一些承认协议,例如专业服务标准协议,过度限制性的原产地规则已证明是有问题的。 在巴西接受的橙子可以在葡萄牙整个欧盟出售,而在葡萄牙获得执照的巴西工程师或会计师必须满足在欧盟其他地方工作的单独的许可证要求,迫使非欧洲工人经受昂贵而低效的官僚程序,从而阻碍了急需的劳动力流动。
此外,在关税和标准方面,世贸组织规则并不是平等的,虽然它们保护被排除在双边或区域关税协定之外的国家,从而确保一体化市场不会获得额外优势,但保护第三国免受强制性标准协定影响的保障措施很少。
首先,它们可以通过同意不强加限制性原产地规则,让所有国家都从双边相互承认协议中获益。 其次,如果它们确实考虑协调,它们可以赞成原标准中不太严格的标准,除非有可信的证据表明它不会支持相关的监管目标。
如果欧盟和美国作出这两项承诺,世界其他国家可以满怀希望而不是惊恐地跟踪TTIP谈判。
中美不平衡的平衡观
北京 — — 2007年7月前,大多数经济学家都认为全球失衡是全球增长的最大威胁。 人们认为,美国外债净额与GDP之比不断上升 — — 长期经常项目赤字所导致的结果 — — 将大幅遏制资本流入,进而削弱美元,抬高利率,使美国经济陷入危机。
但这一情景并没有实现。 相反,危机源自美国次贷崩溃,美国从1930年代以来迅速将全球经济拖入最深的衰退。
大部分经济学家都未能预见导致危机的经济动态,因为他们没有足够关注美国总债务的快速增长。 相反,他们只关注美国的外债,忽视了家庭债务(抵押和消费债务 ) 、 公共债务、商业债务和金融债务。
特别是,它们本应该更加关注美国按揭和消费债务的可持续性。 2007年,按揭和消费债务与GDP之比超过90%,而净外债为24%。
当然,债务的各个组成部分在性质和融资来源上 — — 因而在可持续性上 — — 差异很大。 但一国总债务的所有部分及其融资方式是相互关联的。 债务的每个部分都相互连接。
第一,不同资金来源的资金在一定程度上可以互换:总债务一个部分的资金不足可以用最初旨在为其他部分提供资金的盈余资金加以补充,第二,总债务任何一个部分的麻烦将对所有其他部分产生影响。
次级贷危机爆发后,按揭和消费债务由家庭以储蓄或违约方式偿付。 美国总债务的下降以及总债务与国内资金之间的融资差距缩小,导致美国2008-2009年经常账户赤字大幅改善。 美联储理事会主席本·伯南克(Ben Bernanke)声称赤字是由全球“储蓄过剩”造成的,他反驳了这一说法。
为了抵消私营部门去杠杆化对增长的负面影响,美国政府保持了扩张性的财政和货币政策。 如今,随着家庭债务在热门政府干预后持续陷入刀尖的境地,财政状况急剧恶化,经常账户收支平衡再次恶化。
公共债务的可持续性取代了私人债务的可持续性,成为金融稳定的最大威胁。 有关美国经常账户的辩论焦点已经从外债的可持续性转向减少对外赤字对增长和就业的影响。 美国决策者所面临的困境是如何刺激增长,同时降低总债务水平。
实现这两个目标的最重要途径是通过加强美国竞争力增加出口。 但提高竞争力将来自何处?
由于快速的财政恶化现在让投资者担心美国政府证券的资本损失,贬值会使外国人更加不愿为美国预算赤字融资。 如果外国融资不到位,美国政府债务的收益率将上升,美国经济将再次陷入衰退。
从长远看,美国的增长模式必须经历结构性转变,从依赖债务和消费转向依靠美国人高呼的创造力和创新能力。 只有这样,美国才能提高竞争力,让政府既能将私人债务和公债降低到可持续水平,同时又能保持可尊重的增长率。
短期而言,美国经常项目赤字将保持不变,不管哪个国家拥有双边盈余。 因此,中国继续将其经常项目盈余再投资于美国政府证券对于美国增长和金融稳定至关重要。
考虑到美国从中国购买美国政府证券中获得巨大好处,很难理解为什么美国政府和国会对双边经常项目赤字抱怨如此之大。 也很难理解为什么中国如此不愿意削减双边盈余,因为其大规模持有美国政府证券的收益微乎其微,未来资本损失的风险也非常大。
好消息是,在胡锦涛主席最近访问华盛顿之后,美国和中国都在采取积极步骤解决双边经常账户收支平衡分歧。 这预示着就全球失衡问题进行更加理性和建设性的中美对话,这肯定会有利于全球经济。
银行联盟婴儿步
布鲁塞尔 — — 金融危机开始时,人们说银行在查尔斯·古德哈特(Charles Goodhart)的尖锐词句中是“生命中的国际,但死亡中的是国民 ” 。 到了那时(2008-2009年),大型国际银行在遭遇麻烦时必须由母国政府拯救。
比如,在西班牙,地方储蓄银行(cajas ) 资助了一场巨大的房地产繁荣。 随着繁荣走向萧条,损失有可能压倒西班牙国家的能力,问题成为欧洲问题,因为它威胁到欧元的生存。
国家监督员总是倾向于将国内问题降到最低程度。 他们的本能(和官僚利益 ) , 是捍卫他们国家的海外“国家冠军”银行。
直至最近,西班牙当局一直认为,西班牙房地产部门的问题只是暂时的。 承认真相意味着承认多年来,他们忽视了不可持续的建筑繁荣的兴起,这一繁荣现在有可能使整个国家破产。
以爱尔兰为例,情况最初没有多大不同。 当问题开始浮现时,财政部长最初声称爱尔兰将进行“有史以来最廉价的银行救助 ” 。 英国的银行救助将达到“前所未有的最低水平 ” 。
鉴于国家监督员的可预见趋势是不承认国内问题,清理破产银行的成本也应该在国家一级承担,这似乎很自然。 因此,即使在欧元区,银行监督在很大程度上仍然是全国性的,这似乎有道理。
但现实表明,这一方针是站不住脚的。 问题可能来自国家一级,但由于货币联盟,问题很快威胁到整个欧元区银行系统的稳定。
在其六月份峰会上,欧洲领导人最终认识到需要纠正这种状况,将欧元区银行监督责任移交给欧洲央行。 鉴于金融一体化在货币联盟中特别强大,让欧洲央行负责是一个显而易见的选择。
此外,欧洲央行已经对欧元区银行系统的稳定性承担了事实上的责任。 但直到现在,欧洲央行不得不向银行大量贷款,而不能判断银行的健全性,因为所有这些信息都掌握在国家当局手中。 国家当局嫉妒地看守银行,通常否认问题,直到为时已晚。
由欧洲央行负责也将有助于阻止逐渐解体的进程,这一进程虽然不公开可见,但却非常真实。 问一下总部设在财政紧张的欧元区国家的任何大型国际银行集团。 欧元区国家需要多少银行集团,只要问一下。
考虑一家总部设在意大利,但在德国有一个重要的子公司的银行。 德国的业务自然产生盈余(因为德国的储蓄远远超过平均投资 ) 。
它希望看到“内部资本市场”尽可能多地运作。 在这里,让欧洲央行作为中立的仲裁人来负责这些对立利益也是有道理的。
但是,一方面让欧洲央行负责银行监督解决了一个问题,另一方面却制造了另一个问题:国家当局是否仍然要为它们不再监督的储蓄银行负责?
官方尚未承认这一点,但这往往是欧洲一体化进程的方式:一个领域的一个不完整步骤后来需要相关领域的进一步步骤。
这一渐进式方法在过去运作良好;事实上,今天的欧盟是其产物。 但金融危机并没有让决策者有时间向选民解释为什么一个步骤需要另一个步骤。
" 安倍经济学 " 的新承诺
东京—日本自由民主党在12月14日的议会选举中赢得了决定性的胜利,日本选民展示了他们对首相安倍晋三宏观经济政策议程的压倒性赞同。 选民投票率虽然相对较低,但主要由于问题的技术性质,选举信息非常明确:大多数日本人憎恶回到日本“基因组学”之前的严峻经济轨迹的前景。
安倍经济学的第二支箭 — — 货币宽松 — — 强化了这些效应。 当安倍经济学的第一个“箭头 ” — —财政刺激计划 — — 近两年前启动时,资产市场的即时反应是积极的。 安倍经济学的第二支箭头 — — 货币宽松 — — 强化了这些效应。
过去两年,日本股市价值几乎翻了一番,增加了日本消费者的财富。 此外,日元对美元下跌了近三分之一,从80日元兑1美元左右下降到120日元兑1美元左右,刺激了日本的出口产业。
更令人鼓舞的是劳动力市场的发展,劳动力市场的发展与资产市场不同,反映的是结果,而不是预期。 这里的消息也是好消息。 劳动力市场的发展与资产市场不同,反映了结果,而不是期望。
但四月份消费税涨幅 — — 从5%升至8% — — 引发的衰退不能归咎于安倍经济学。 事实上,安倍荣耀日本民主党领导的上届政府颁布的一部法律。
